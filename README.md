# StudenskaSluzba
Program za skladištenje i ispisivanje podataka vezanih za studentsku službu

# Funkcionalnosti:

  1. Pregled svih podataka:
     - Nastavnika
     - Predmeta
     - Studenata
     - Ispitnih Rokova
     - Isputnih Prijava
     - Nastavika i predmeta koje predaju
     - Studenata i predmeta koje pohađaju
       
  2. Dodavanje podataka:
     - Nastavnika
     - Predmeta
     - Ispitnog Roka
     - Ispitne Prijave
